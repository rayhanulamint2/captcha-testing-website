How It Works

A step-by-step guide to the testing process and each CAPTCHA type.

Testing Process

1

Enter Your Participant ID

Go to the Testing Hub and enter the participant ID you were provided. This ID is used only to link your test sessions together — no personal information is collected.

2

Choose a CAPTCHA to Test

Select any of the five CAPTCHA types to test. You can complete them in any order you prefer. A progress tracker shows which ones you've completed.

3

Complete the Challenge

Each CAPTCHA presents a different type of challenge. Follow the on-screen instructions and complete the verification. Your interactions are automatically recorded.

4

Return and Test More

After completing one CAPTCHA, return to the Testing Hub and try another. We encourage you to complete all five for the most useful data.

5

Done!

Once you've tested all five CAPTCHAs, you're done! Your anonymous data will be included in our research analysis. Thank you for contributing!

CAPTCHA Types Explained

✏️

DrawCAPTCHA Novel

Behavioral path-tracing verification

How it works:

  1. A guided path is displayed between a green start dot and a red end dot
  2. A brief animation shows you the path to trace
  3. Using your mouse or finger, draw along the dashed guide path
  4. Click "Verify" to submit your drawing
  5. Both path accuracy AND behavioral patterns are analyzed

What we measure:

  • Path accuracy (average distance from guide)
  • Drawing speed and timing patterns
  • Hand tremor (jitter) — a human indicator
  • 16 behavioral metrics for bot detection

Difficulty levels:

Paths range from simple (zigzag, arc) to complex (spiral, infinity, interference). Each has an easy, medium, or hard difficulty rating.

🖼️

Image CAPTCHA Traditional

Visual object recognition grid

How it works:

  1. A 3×3 grid of images is displayed
  2. You are asked to select all images matching a category (e.g., "Select all images with traffic lights")
  3. Click on matching images to select/deselect them
  4. Click "Verify" to submit your selections

What we measure:

  • Selection accuracy
  • Time to complete
  • Number of incorrect selections
🔊

Audio CAPTCHA Traditional

Spoken word recognition under noise

How it works:

  1. Click the play button to hear an audio clip
  2. Listen carefully — spoken words are mixed with background noise
  3. Type the words you heard into the text field
  4. Click "Verify" to check your answer
  5. You can replay the audio if needed

What we measure:

  • Transcription accuracy
  • Number of replays
  • Time to complete
🔤

Text CAPTCHA Traditional

Distorted text recognition

How it works:

  1. A canvas displays distorted, warped text with noise
  2. Read the characters despite visual distortion
  3. Type what you see into the input field
  4. Click "Verify" to check (case-insensitive)
  5. Click "New" to get a different text if it's too hard to read

What we measure:

  • Character-level accuracy
  • Solve time
  • Number of refreshes
🎮

Game CAPTCHA Experimental

Interactive puzzle challenge — the hardest test

How it works:

  1. A jigsaw-style puzzle is presented with scrambled pieces
  2. Drag pieces to their correct positions on the board
  3. Some pieces may need to be rotated (click to rotate)
  4. Complete the puzzle within the time limit
  5. This is intentionally the most challenging CAPTCHA

What we measure:

  • Completion time
  • Number of moves
  • Accuracy of placements
  • Spatial reasoning performance

What Data We Collect

✅ We DO Collect

  • Participant ID (anonymous)
  • Session ID (random)
  • CAPTCHA solve time
  • Accuracy / pass/fail result
  • Mouse/touch movement data
  • Browser type and OS
  • Screen resolution
  • Number of attempts
  • Behavioral metrics (for DrawCAPTCHA)

❌ We DO NOT Collect

  • Your name or email
  • IP address
  • Location data
  • Cookies or tracking pixels
  • Any personal identifying information
  • Browsing history
  • Data from other tabs/windows
