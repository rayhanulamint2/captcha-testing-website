Five CAPTCHA Types Under Study

Each CAPTCHA type is tested for solve time, accuracy, user satisfaction, and bot resistance.

✏️

DrawCAPTCHA

Behavioral

Trace a guided path on a canvas. A behavioral CAPTCHA that uses 16 anti-bot checks including jitter analysis, velocity variance, and timing patterns.

  • Canvas-based path tracing
  • 25 unique path shapes
  • Behavioral biometrics
🖼️

Image CAPTCHA

Traditional

The familiar "select all images with..." grid challenge, similar to Google reCAPTCHA v2. Users identify objects in a grid of images.

  • 3×3 image grid selection
  • Object recognition task
  • Multiple categories
🔊

Audio CAPTCHA

Traditional

Listen to spoken words with background noise and type what you hear. Tests auditory processing under distortion.

  • Spoken word recognition
  • Background noise layers
  • Replay functionality
🔤

Text CAPTCHA

Traditional

Classic distorted text that users must read and type. Includes warping, noise lines, and color distortion on a canvas.

  • Distorted text rendering
  • Noise & line overlays
  • Case-insensitive matching
🎯

GotCHA CAPTCHA

Novel

Find sketched icons hidden within a real-world image and click them in the correct order. A visual search challenge testing pattern recognition.

  • Sketch-match icon search
  • Sequential order selection
  • Visual pattern recognition

Why This Research Matters

🤖

Bot Evolution

Automated bots are increasingly sophisticated. AI can now solve traditional CAPTCHAs with high accuracy, demanding new approaches.

♿

Accessibility Gap

Many CAPTCHAs exclude users with disabilities. Our study measures accessibility across visual, auditory, and motor dimensions.

⏱️

User Frustration

CAPTCHAs that are too hard reduce conversion rates. We measure solve time, error rates, and subjective satisfaction.

🔬

Empirical Data

Instead of theoretical comparisons, we collect real interaction data from participants to draw evidence-based conclusions.
